Find and replace all the strings from all the fields of all the tables.

== Description ==

Find and replace all the strings from all the fields of all the tables. This plugin takes auto backup of the full database. The MySql user must have the required permissions. It's always recommended to take a manual backup for safer side.

== Installation ==

This section describes how to install the plugin and get it working.This section describes how to install the plugin and get it working.

1. Download the plugin archive file.
2. Expand it to wp-content/plugins/ directory or can be uploaded the archive file from WordPress backend.
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Is is case sensitive? =

Yes

= Where are the backups stored? =

Backups are stored in wp-content/backups/db-backup-[Y-m-d-His].sql
